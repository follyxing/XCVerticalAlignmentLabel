# Example
```objective-c
XCVerticalAlignmentLabel * label = [[XCVerticalAlignmentLabel alloc]initWithFrame:CGRectMake(0, 0,100 , 100)];
[label setText:@"hello world!"];
[label setVerticalAlignment:XCVerticalAlignmentBottom]; //default top
```
